### Css: Cheat Sheet

#### Reset
```
* {                                                                             
  margin: 0px;                                                                
  padding: 0px;                                                               
  box-sizing: border-box;                                                     
}                                                                               
```

#### Text properties
```
  font-family: "Lato", sans-serif;
  font-weight: 400; /* text thick or thin */
  font-size: 16px; 
  line-height: 1.7; /* space between text lines
  color: #777; /* text color */                                                                            
```
